# Screenshots

사용자 가이드에서 참조하는 스크린샷을 이 폴더에 저장합니다. 파일명이 문서의 참조와 일치해야 이미지가 연결됩니다.

## 필요한 스크린샷 목록

앱 실행·캡처는 Xcode 시뮬레이터 또는 실기기에서 Cmd+S / 음량+상단 버튼 동시 누르기로 가능합니다.

### 앱 전반
| 파일명 | 내용 |
|--------|------|
| `home.png` | 홈 화면 전체 — 타이틀바 + 채널 헤더(접힘) + 시그널 체인 + 이펙트 에디터 + 루퍼 바 |
| `channel-expanded.png` | 채널 헤더 펼쳐진 상태 — 타입 토글 + Gain/Pan 슬라이더 |
| `settings.png` | ⚙ 설정 시트 — 버퍼, 샘플레이트, 백그라운드 오디오, 물리 채널 override |

### 루퍼
| 파일명 | 내용 |
|--------|------|
| `looper-collapsed.png` | 하단 접힌 루퍼 바 |
| `looper-expanded.png` | 펼친 루퍼 상세 패널 (2슬롯, 진행바, 컨트롤) |

### MIDI

## 촬영 팁

- iPhone 15 Pro (6.1") 또는 iPad mini 기준 해상도 권장
- 다크 모드 고정 (앱이 `.preferredColorScheme(.dark)` 강제)
- 상태바는 깔끔한 시간·풀 배터리가 이상적
- 각 이펙트 스크린샷은 **에디터 카드 전체가 보이도록** 시그널 체인 바까지 포함
