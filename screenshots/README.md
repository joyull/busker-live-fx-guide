## Capture guidelines

- iPhone 15 Pro (or the same device across all shots) for visual consistency
- Dark mode enforced (the app uses `.preferredColorScheme(.dark)`)
- Keep the status bar clean — tidy time, full battery
- For effect shots, keep the signal-chain bar visible above the editor card
- Export as PNG, sRGB, resized to ~1200 px wide. Target <500 KB per file
