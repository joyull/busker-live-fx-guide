# 루퍼 사용법

2슬롯 루프 레코더. 화면 하단에 항상 바가 떠 있고, 탭하면 상세 패널이 펼쳐집니다.

## 화면 구조

![루퍼 바 접힘](../screenshots/looper-collapsed.png)
<!-- SCREENSHOT: 하단 접힌 루퍼 바 — Looper 타이틀 + A/B 슬롯 상태 도트 + ON/OFF 토글 -->

```
🌀 Looper  ⌄   ●A  ○B                     [ ● ON ]
```

- **⌄ / ⌃ 화살표**: 상세 패널 펼침/접힘
- **A / B 도트**: 각 슬롯 상태 색상 (빨강=녹음, 초록=재생, 주황=오버더빙, 회색=Idle)
- **ON/OFF 토글**: 루퍼 자체 활성화. OFF면 DSP 체인에서 제외

![루퍼 바 펼침](../screenshots/looper-expanded.png)
<!-- SCREENSHOT: 펼친 루퍼 상세 패널 — 2개 슬롯, 진행바, 컨트롤 버튼 -->

펼친 상세 패널에는 슬롯 A와 B가 각각 세로로 배치됩니다.

```
┌──────────────────────────────────────────┐
│  ●  Loop A   ▶ PLAY                4.2s │
│  ▓▓▓▓▓▓▓▓░░░░░░░░░░░░░░░                │  ← 재생 위치 진행바
│  [REC] [PLAY] [OVR] [STOP]  UNDO  CLR   │
├──────────────────────────────────────────┤
│  ●  Loop B   ■ IDLE                 --  │
│  ░░░░░░░░░░░░░░░░░░░░░░░░░░░            │
│  [REC] [PLAY] [OVR] [STOP]  UNDO  CLR  SYNC │
└──────────────────────────────────────────┘
```

## 슬롯 상태

| 상태 | 색상 | 의미 |
|------|------|------|
| **Idle (■)** | 회색 | 녹음된 내용 없음 |
| **Recording (●)** | 빨강 | 현재 녹음 중 (입력 → 버퍼 기록) |
| **Playing (▶)** | 초록 | 녹음된 루프 재생 중 |
| **Overdub (◉)** | 주황 | 재생하면서 입력을 추가 기록 중 |

## 컨트롤 버튼

### REC — 녹음
- **Idle에서 누르면**: 녹음 시작 (빨강).
- **Recording에서 누르면**: 녹음 종료 후 자동 **Playing** 으로 전환.
- 최대 길이: **약 60초**. 넘으면 자동으로 Playing 전환.

### PLAY — 재생
- 녹음된 내용을 처음부터 재생. 이미 재생 중이면 위치 유지.
- **Overdub에서 누르면**: 추가 기록 중지, 순수 재생 모드로.

### OVR — 오버더빙
- **Playing 상태에서만 동작**. 재생하면서 입력 신호를 누적 기록.
- 다시 누르면 Playing으로 돌아감.
- 같은 루프를 여러 레이어로 쌓을 수 있음.

### STOP — 정지
- 어떤 상태에서든 즉시 Idle에 가까운 정지 상태로. 녹음된 내용은 보존.
- 다음번 PLAY를 누르면 처음부터 다시 재생.

### UNDO — 마지막 오버더빙 되돌리기
- 가장 최근에 쌓은 오버더빙 레이어 하나를 제거. 원본 녹음은 유지.

### CLR — 전체 지우기
- 해당 슬롯의 모든 녹음 내용 삭제 → Idle로 복귀.

### SYNC — 길이 동기화 (Loop B 전용)
- 슬롯 B를 녹음할 때 슬롯 A의 길이에 맞춰 자동 종료.
- 예: A가 4.0초짜리 루프일 때 B에서 녹음을 시작하면, 4.0초가 되는 순간 자동으로 Playing 전환.
- 주의: A가 녹음되어 있지 않으면 SYNC는 무시됩니다.

## 사용 예시

### 기본 루프 만들기
1. 루퍼 바의 **ON/OFF 토글**로 루퍼 활성화.
2. **Loop A** → REC 탭 → 기타 연주 → 다시 REC 탭 (자동으로 재생 시작).
3. 재생되는 루프 위에 기타로 연주 추가 → OVR 탭 → 다시 OVR 탭 해서 종료.
4. 정지하려면 STOP, 지우려면 CLR.

### 두 레이어 겹치기 (A + B)
1. **Loop A**: 리듬 패턴 녹음 (REC → REC).
2. **Loop B**: SYNC 활성화 → REC → A의 길이와 자동으로 맞춰짐.
3. B에 리드나 코드를 녹음 → 재생 위치가 A와 동일하게 동기.

### 실시간 오버더빙
1. A에 베이스 라인 녹음.
2. 재생되는 A 위에서 OVR → 하모니 연주 → OVR 해제.
3. 마음에 안 들면 **UNDO**로 방금 레이어만 제거.

## MIDI 풋컨트롤러로 제어

MIDI 매핑을 설정하면 풋스위치로 REC / PLAY / OVR / STOP / UNDO / CLR을 조작할 수 있습니다. 자세한 설정은 [MIDI 가이드](midi.md) 참고.

대표 매핑 예:
- 풋스위치 1 → **Looper Record (Loop A)** — 녹음 시작/정지
- 풋스위치 2 → **Looper Overdub (Loop A)**
- 풋스위치 3 → **Looper Stop (Loop A)**

## 제약 및 팁

- 루프는 **현재 세션 메모리에만** 저장됩니다. 앱 종료 시 사라지므로 중요한 녹음은 외부 레코딩을 병행하세요.
- 루퍼 신호는 **모든 시그널 체인 이펙트 다음**에 들어갑니다 (녹음된 내용은 이펙트가 걸린 완성된 사운드).
- Loop A와 Loop B는 **완전 독립** — 서로 다른 템포·길이로 재생해도 무방 (SYNC를 끄면).
- Overdub 레이어 깊이는 UNDO 한 단계만 가능 (최근 레이어 하나만 되돌리기). 여러 단계 되돌리려면 CLR 후 재녹음.
