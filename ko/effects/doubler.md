# Doubler — 더블러

기타 트랙을 **여러 번 녹음한 것처럼** 두툼하고 넓게 만드는 이펙트. 어쿠스틱 기타 스테레오 레코딩 기법을 실시간으로 흉내냅니다.

## 화면 구성

```
┌───────────────────────────────────────────────┐
│  Doubler                           [ ON ]     │
├───────────────────────────────────────────────┤
│   [ Voices: 1 | 2 ]                 [Expert]  │
│                                                │
│   🎛 Mix    🎛 Tight   🎛 Width  🎛 HF Tilt   │
│                                                │
│   ─── Expert 전용 (켰을 때만) ─────────────    │
│   🎛 Detune  🎛 Spread  🎛 Jitter             │
└───────────────────────────────────────────────┘
```

## 메인 파라미터

| 파라미터 | 범위 | 설명 |
|---------|------|------|
| **Voices** | 1 / 2 | 추가 레이어 개수. 1=모노 더블, 2=스테레오 더블 |
| **Mix** | 0–100 % | Dry와 Doubled의 블렌드 |
| **Tight** | 0–100 % | 더블된 레이어의 타이밍 정확도. 높을수록 오리지널에 가까움 |
| **Width** | 0–100 % | 스테레오 폭 (Voices=2일 때 의미) |
| **HF Tilt** | −3 to +3 dB | 더블링된 레이어의 고역 조정 |

## Expert 파라미터

Expert 스위치를 켜면 세밀한 조정용 3개 노브가 추가됨.

| 파라미터 | 범위 | 설명 |
|---------|------|------|
| **Detune** | ±0–25 cents | 레이어 피치 변조 정도 |
| **Spread** | 0.5× to 2.0× | 레이어 간 시간 간격 배율 |
| **Jitter** | 0–100 % | 랜덤 변동 추가 — "사람 손" 느낌 |

## 사용 예시

### 기본 스테레오 더블 (포크/팝 레코딩)
- Voices **2**, Mix 35%, Tight 60%, Width 80%, HF Tilt +1 dB
- 스테레오 필드가 자연스럽게 확장되며 중심 이미지는 유지

### 미묘한 두께 추가
- Voices **1**, Mix 20%, Tight 80%, Width 0%, HF Tilt 0 dB
- 모노 유지하면서 소리만 두툼하게

### 과감한 레이어링 (앰비언트·워십)
- Voices 2, Mix 50%, Tight 30%, Width 100%, HF Tilt +2 dB, Expert ON
- Detune ±15c, Spread 1.5×, Jitter 40%
- 여러 기타가 동시에 연주하는 듯한 느낌

### 주의할 조합
- Tight 0% + Detune ±20c 이상: 너무 "out of tune" 같은 소리
- Width 100% + 1 Voice: Width가 무의미 (스테레오 레이어가 없음)
- Jitter 80%+: 과도하게 흔들려 부자연스러움

## Voices 1 vs 2

- **1 Voice (모노 더블)**: 원본 + 레이어 1개. Width는 무의미. 스테레오 입력이면 레이어가 양쪽에 살짝 퍼짐.
- **2 Voices (스테레오 더블)**: 원본 + 좌/우 레이어 2개. Width가 커질수록 좌우 분리.

## 신호 순서 팁

Doubler는 **IR Loader 다음, Chorus 이전** 위치. 톤이 완성된 뒤에 공간을 넓히는 단계라고 생각하시면 됩니다. Reverb와 함께 쓸 때는 Doubler를 먼저 → Reverb 나중 순서가 자연스러워요.
