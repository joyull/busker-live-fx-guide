# Chorus — 코러스

미세한 피치·시간 변조로 소리를 **움직이게 만드는** 이펙트. 어쿠스틱 기타에서는 잔잔한 공간감이나 "젖은" 질감을 위해 사용합니다.

### 기본 모드
### Expert 모드
## Mode (1–4)

4가지 내부 알고리즘 프리셋. 숫자가 커질수록 더 풍부한 변조 특성.

| Mode | 캐릭터 | 추천 용도 |
|------|--------|-----------|
| **1** | 클래식 단일 LFO, 부드러움 | 잔잔한 백킹 |
| **2** | 2개 LFO 스테레오, 넓음 | 일반적인 코러스 톤 |
| **3** | 복합 LFO, 풍부 | 앰비언트 핑거스타일 |
| **4** | 깊은 변조, 스페이시 | 실험적 사운드 |

## 기본 파라미터

| 파라미터 | 범위 | 설명 |
|---------|------|------|
| **Mix** | 0–100 % | Dry ↔ Chorus 블렌드 |
| **Width** | 0–100 % | 스테레오 폭 |
| **Character** | 0–100 % | 톤 캐릭터 (0=밝고 얇음, 100=두껍고 어두움) |

## Expert 파라미터

Expert를 켜면 LFO를 직접 조정할 수 있습니다.

| 파라미터 | 범위 | 설명 |
|---------|------|------|
| **Rate** | 0.05–1.5 Hz | LFO 주기 (속도) |
| **Depth** | 0–100 % | 변조 깊이 |

Mode 버튼은 Expert에서 숨겨지지만 내부 값은 유지됩니다.

### 잔잔한 앰비언스
- Mode **2**, Mix 20%, Width 60%, Character 30%
- 기타 뒤에 얇게 걸어 공간감 추가

### 80년대 팝 스트러밍
- Mode **3**, Mix 40%, Width 100%, Character 60%
- 선명한 스테레오 움직임

### 앰비언트 핑거스타일
- Mode **4**, Mix 35%, Width 80%, Character 70%
- Expert: Rate 0.3 Hz, Depth 50%
- 느리고 깊은 변조로 몽환적

### 거의 들리지 않는 보정용
- Mode 1, Mix 10%, Width 30%, Character 0%
- 존재감 살짝 더하고 싶을 때

## Rate와 Depth의 조합 (Expert)

| 조합 | 느낌 |
|------|------|
| Rate 느림 + Depth 얕음 | 거의 안 들림 (존재감 미묘) |
| Rate 느림 + Depth 깊음 | 느린 워블, 테이프 wow 느낌 |
| Rate 빠름 + Depth 얕음 | 선명한 기본 코러스 |
| Rate 빠름 + Depth 깊음 | 비브라토 / 매우 liquid한 톤 |

일반적인 어쿠스틱 코러스는 Rate **0.3–0.8 Hz**, Depth **30–60%** 범위가 자연스럽습니다.

## 신호 순서 팁

Chorus는 Doubler 다음, Delay 이전에 위치. Chorus가 만들어낸 변조가 Delay에 반복되면 더 풍부한 공간감이 생깁니다.
