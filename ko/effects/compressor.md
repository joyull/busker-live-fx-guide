# Compressor — 컴프레서

큰 소리는 누르고 작은 소리는 덜 누르는 다이내믹 프로세서. 어쿠스틱 기타에서는 핑거피킹의 어택을 균일하게 만들어 녹음·라이브 믹스에서 잘 들리게 합니다.

![Compressor 화면](../../screenshots/effect-compressor.png)

## 화면 구성

```
┌────────────────────────────────────────────┐
│  Compressor                     [ ON ]     │
├────────────────────────────────────────────┤
│    🎛 Comp     🎛 Level     🎛 Tilt        │  ← Row 1
│    🎛 Mids   [ Fast | Slow ]  🎛 Blend     │  ← Row 2
└────────────────────────────────────────────┘
```

## 파라미터

| 파라미터 | 범위 | 설명 |
|---------|------|------|
| **Comp** | 0–100 % | 압축 강도. 높을수록 어택이 부드럽고 서스테인이 길어짐 |
| **Level** | 0–20 dB | 압축 후 메이크업 게인 (출력 레벨 보정) |
| **Tilt** | −6 to +6 dB | 저역-고역 기울기 조정 (+ = 밝게, − = 따뜻하게) |
| **Mids** | −10 to +10 dB | 중역대 부스트/컷 (존재감 조정) |
| **Attack** | Fast / Slow | 트랜지언트 반응 속도 — Fast는 빠른 어택 제어, Slow는 어택 살림 |
| **Blend** | 0–100 % | 압축 후 신호와 원본 dry의 병렬 믹스 (패러럴 컴프레션) |

## 조작 가이드

### 핑거피킹 (기본)
- Comp 40–60%, Level 4 dB, Attack **Slow**, Blend 70%
- 어택은 살리고 서스테인을 늘려 **플럭**이 또렷하게 들림

### 스트러밍
- Comp 30–50%, Level 2 dB, Attack **Fast**, Blend 100%
- 피크를 눌러 전체 볼륨 균일화. Blend를 낮추지 않는 게 좋음

### 라이브 안전 설정
- Comp 70–80%, Level 6 dB, Attack Fast, Blend 100%, Tilt −1 dB
- 피드백 위험 완화 + 앰프 입력 레벨 안정화

### Tilt와 Mids를 언제 쓰나
- **Tilt**: 컴프레션 후 전체 톤이 어두워지면 **+2 dB**로 밝기 복원
- **Mids**: 믹스에서 어쿠스틱이 묻히면 **+3 dB** (500 Hz~1 kHz 영역 강조)

## Blend의 의미

Blend 100% = 압축된 신호만 출력. Blend 70% = 압축 70% + 원본 30%의 **병렬 믹스**.

병렬 컴프레션은 **원래 다이내믹을 유지하면서 서스테인만 보강**하는 기법이라, 어쿠스틱 기타처럼 자연스러움이 중요한 악기에 자주 쓰입니다.
